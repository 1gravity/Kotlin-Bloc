The core class of the Bloc framework. (can't be an interface or we won't have the generics in Swift)

A Bloc is also a BlocState (allowing us to use a Bloc as BlocState and create composable Blocs):

  • a StateStream emitting state

  • a Sink accepting actions that might trigger state changes

A Bloc is also

  • a SideEffectStream emitting side effects that can be used e.g. for navigation
