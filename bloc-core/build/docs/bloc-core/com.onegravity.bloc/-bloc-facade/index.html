interface BlocFacade<out State, in Action, out SideEffect> : StateStream<State> , Sink<Action>
Content copied to clipboard

The BlocFacade is the "public" part of a Bloc. It defines:

  • a StateStream emitting state

  • a Sink accepting actions that might trigger state changes

  • a SideEffectStream emitting side effects that can be used e.g. for navigation
